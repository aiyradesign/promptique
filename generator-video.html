Cara Mendapatkan API Key

Untuk menggunakan fitur AI, Anda memerlukan Google AI API Key. Ikuti langkah-langkah berikut:

  1. Buka Google AI Studio.
  2. Login dengan akun Google Anda.
  3. Klik tombol Get API key.
  4. Klik Create API key in new project.
  5. Salin API key yang muncul, lalu tempel di aplikasi ini.

Pastikan untuk menjaga kerahasiaan API Key Anda dan tidak membagikannya kepada siapapun.
