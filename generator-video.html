Panduan API Key

Untuk menggunakan fitur AI, Anda memerlukan Google AI API Key. Ikuti langkah-langkah berikut:

  1. Klik tombol "Dapatkan Key" untuk membuka Google AI Studio.
  2. Login dengan akun Google Anda.
  3. Klik tombol Get API key.
  4. Klik Create API key in new project.
  5. Salin API key yang muncul, lalu tempel di aplikasi ini dan klik "Simpan".

Jaga kerahasiaan API Key Anda dan jangan bagikan kepada siapapun.
